# IT Asset Management System

## Overview

A Java-based IT Asset Management System designed to manage company technology assets.

The project demonstrates object-oriented programming concepts including:

- Abstraction
- Inheritance
- Polymorphism
- Encapsulation
- ArrayList collections


## Features

✅ Add network devices  
✅ Add security systems  
✅ Search assets  
✅ Update assets  
✅ Remove assets  
✅ Generate company reports  
✅ Calculate asset values  
✅ Identify highest and lowest value assets  


## Technologies Used

- Java
- Object-Oriented Programming
- ArrayList
- Exception Handling


## Class Structure

ITAsset (Abstract Class)

↓

NetworkDevice

SecuritySystem


## How To Run

1. Clone repository

2. Open project in Java IDE

3. Run Main.java
